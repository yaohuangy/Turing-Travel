# 2026 桂林山水深度游玩全攻略

## 1. 目的地简介

桂林位于广西壮族自治区东北部，以“山水甲天下”闻名世界。这里是典型的喀斯特地貌，漓江穿城而过，两岸奇峰林立，构成“山清、水秀、洞奇、石美”的绝美画卷。桂林气候温和，四季皆宜游览，每年4月至10月雨量充沛，漓江水量充足，是最佳观景季节。适合摄影爱好者、自然风光迷、亲子家庭和文化体验者。

## 2. 核心景点推荐 (含门票与位置信息)

### 2.1 漓江精华段 (杨堤至兴坪)

* **位置**：桂林市阳朔县（从桂林市区乘车至杨堤码头约1小时）
* **门票**：竹筏或游船 215-360元/人（根据船型）
* **游玩时长**：半天至1天
* **简介**：漓江最精华的20公里水道，九马画山、黄布倒影（20元人民币背景）、兴坪古镇均在此段。乘竹筏顺流而下，两岸青山如画，是桂林山水的灵魂体验。建议选上午时段，光线最好。注意：水位过高或过低时竹筏可能停航，需提前查询。

### 2.2 阳朔西街

* **位置**：桂林市阳朔县城中心
* **门票**：免费
* **游玩时长**：2-3小时
* **简介**：有1400多年历史的古街，中西文化交融的独特街区。白天安静古朴，夜晚灯火辉煌，酒吧、咖啡馆、手工艺品店热闹非凡。啤酒鱼是这里的招牌美食，随处可见。

### 2.3 十里画廊

* **位置**：阳朔县城南面，沿321国道
* **门票**：免费（沿途小景点另收费）
* **游玩时长**：半天
* **简介**：长约10公里的景观公路，沿途有月亮山、大榕树、蝴蝶泉等景点。最适合租电动车或自行车游览，走走停停，欣赏田园风光与喀斯特峰林交织的画卷。骑行是阳朔最经典的体验之一。

### 2.4 象鼻山

* **位置**：桂林市象山区滨江路1号
* **门票**：55元/人
* **游玩时长**：1-2小时
* **简介**：桂林城徽，山形酷似大象饮水，是桂林最标志性的景观。公园不大，可登山俯瞰桂林市区，也可在山脚水月洞拍照。建议清晨或傍晚前往，光线柔和，游人较少。

### 2.5 龙脊梯田

* **位置**：桂林市龙胜各族自治县（距市区约2.5小时车程）
* **门票**：80元/人
* **游玩时长**：1-2天
* **简介**：世界农业文化遗产，规模宏大的梯田群分布在海拔300-1100米之间。主要观景区域有金坑大寨和平安壮寨，5-6月灌水季镜面效果最震撼，9-10月稻谷金黄层林尽染。建议住一晚，看日出晨雾。注意：山路弯曲，晕车需备药。

## 3. 特色餐饮与预算参考

* **桂林米粉**：桂林人的一日三餐，卤水配锅烧肉、卤牛肉，干捞为最正宗吃法，人均预算 **8-15元**。
* **啤酒鱼**：阳朔招牌菜，漓江鲜鱼配啤酒番茄焖制，人均预算 **60-100元**。
* **荔浦芋扣肉**：桂林传统名菜，粉糯芋头配五花肉，人均预算 **45-65元**。
* **田螺酿**：阳朔特色，将螺肉与猪肉剁碎后回填螺壳烹制，人均预算 **35-55元**。
* **艾叶粑粑**：桂林传统小吃，糯米裹艾叶蒸制，清甜软糯，人均预算 **5-10元**。
* **油茶**：桂北地区特色，茶叶姜蒜捶打后煮成的浓汤，配米花、花生，人均预算 **10-20元**。

## 4. 住宿区域建议

* **桂林市区/两江四湖周边**：交通便利，靠近象鼻山、日月双塔，逛街吃饭方便。酒店预算：**200-500元/晚**。
* **阳朔西街周边**：热闹方便，出门就是餐饮酒吧，适合喜欢夜生活的游客。客栈预算：**200-450元/晚**。
* **遇龙河沿岸**：安静田园风光，推开窗就是山水画卷，适合度假放松。民宿预算：**350-800元/晚**。
* **龙脊梯田景区内**：住在瑶寨/壮寨木楼，方便看日出晨雾。民宿预算：**150-400元/晚**。

## 5. 经典三日行程参考

* **第一天：桂林城市印象**
  * 上午：游览【象鼻山公园】，登山俯瞰漓江与城市风光。
  * 下午：漫步【两江四湖】景区，参观日月双塔、古南门。
  * 晚上：在正阳步行街或东西巷品尝桂林米粉，看杉湖夜景。
* **第二天：漓江精华与阳朔风情**
  * 上午：乘车至杨堤码头，乘竹筏游览【漓江精华段】至兴坪，打卡20元背景。
  * 下午：从兴坪乘车至阳朔，入住后租电动车骑行【十里画廊】。
  * 晚上：逛【阳朔西街】，品尝啤酒鱼，感受中西合璧的夜生活。
* **第三天：田园风光与山水秘境**
  * 上午：骑行或乘竹筏游览【遇龙河】，体验人工竹筏的宁静与诗意。
  * 下午：探索月亮山、大榕树等沿途小景，或前往富里桥拍古桥倒影。
  * 晚上：在西街做最后的采购和美食，结束行程返程。
